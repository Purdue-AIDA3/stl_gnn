# STL-GNN: Learning-Guided STL-based Path Planning

This repository contains code for a learning-guided approach to solving signal temporal logic (STL)-based path planning problems using graph neural networks (GNNs).

We learn strategic binary variables arising from the logical and temporal structure of STL specifications, enabling efficient convex optimization during inference.
